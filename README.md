# ✤ abdmmar.com

Personal site. Built using [Astro](https://astro.build/) and [IBM Plex](https://www.ibm.com/plex/) typeface 

## ✦ Development

```bash
# clone project
git clone https://github.com/abdmmar/blog.git
cd blog

# install dependencies
yarn install

# build project
yarn dev
```
