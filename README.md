# ✤ abdmmar.com

Personal site. Built using [Astro](https://astro.build/) and [Inter](https://rsms.me/inter/) typeface 

## ✦ Development

```bash
# clone project
git clone https://github.com/abdmmar/blog.git
cd blog

# install dependencies
pnpm install

# build project
pnpm dev
```
